# Elder Mind

Un gioco incrementale / clicker narrativo.

## 🎮 Gioca qui
https://ivotto-cap.github.io/elder-mind/

## 🧠 Descrizione

Interpreti un cervello sintetico che si risveglia e inizia a generare impulsi.

Attraverso upgrade, eventi e interazioni, il gioco evolve fino a rivelare la sua vera natura.

## ⚙️ Tecnologie

- HTML
- JavaScript
- Audio dinamico
- Sistema di eventi

## 🚧 Stato

Demo in sviluppo

## 💡 Note

Il gioco è progettato per essere giocato attivamente, non solo in idle.
